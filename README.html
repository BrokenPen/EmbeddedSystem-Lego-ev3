Lego EV3 as STEM educational tool in school

Lego mindstorms NXT[1] , Lego mindstorms EV3[2] are popular tools that using in school as a STEM educational tool. To teaching children how to programming via the Lego official mindStorms program[3] program with visual programming language[4]
